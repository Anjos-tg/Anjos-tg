## 🚀 Sobre mim

Desenvolvedor focado em **automação de processos, sistemas empresariais e soluções tecnológicas para ambientes industriais**.

Crio ferramentas que conectam **gestão, produção e dados**, aumentando a eficiência operacional e permitindo maior rastreabilidade e controle de processos.

Tenho experiência em desenvolvimento de sistemas baseados no **Frappe Framework**, construção de APIs, automação de infraestrutura e integração entre diferentes plataformas.
